# AI-Based Personalized Learning Path Recommendation System

## Project Overview
This project implements an AI-based personalized learning system that analyzes student performance data and recommends adaptive learning paths based on individual strengths and weaknesses.

## Key Features
- Machine Learning-based learning level classification
- Personalized content recommendation
- Interactive student performance evaluation
- Overall performance summary (weak/strong topics)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook
